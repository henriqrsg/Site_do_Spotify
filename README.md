# Spotify Clone Website

Projeto de um site inspirado na interface do Spotify, desenvolvido utilizando **HTML5**, **CSS3** e **JavaScript**, com foco em layout moderno, organização visual e reprodução fiel do design original.

## Objetivo

Este projeto foi criado com o intuito de praticar:

* Reprodução de interfaces reais (UI Clone)
* Estruturação de páginas com HTML5 semântico
* Estilização moderna com CSS3
* Construção de layouts com Flexbox e Grid
* Organização visual baseada em aplicações reais

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Sidebar com menu de navegação (Home, Buscar, Biblioteca)
* Área principal com playlists e álbuns
* Seção de artistas e músicas em formato de grid
* Layout inspirado no design original do Spotify
* Interface em modo escuro (dark mode)
* Navegação visual simples e intuitiva

## Destaques Técnicos

* Uso de **Flexbox** para estrutura principal (sidebar + conteúdo)
* Uso de **CSS Grid** para organização de playlists e álbuns
* Separação clara entre estrutura (HTML) e estilo (CSS)
* Organização modular de arquivos
* Padronização de espaçamentos e tipografia
